# PIXELATOR

Pixelator is a rather simple web app for creating pixel art. I took lots of 
ispiration from the likes of [Piskelapp](https://www.piskelapp.com/) and such

## Road map

Features that i wish to implement:

- [ ] Variable sprite size
- [ ] Variable brush size
- [ ] Colors
- [ ] Layers
- [ ] Frames
- [ ] Fill tool
- [ ] Select tool
- [ ] Line tool
